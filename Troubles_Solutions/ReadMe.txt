Android Studio Error
====================
============================================================================
Error 1	: "Run" does not update APK in your mobile even uninstall and re-run

Solution: Disable Instant Run feature as follow
	  File => Settings... => Build, Execution, Deployment => Instant Run
	  Then uncheck the first check box

See	: Disable_Instant_Run.jpg

============================================================================
============================================================================
Error 2	: Unable to render .xml design

Solution: Disable Automatically Pick Best feature

See	: Disable_Auto_Pick.jpg
============================================================================
